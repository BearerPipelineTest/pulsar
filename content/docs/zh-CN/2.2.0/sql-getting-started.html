Pulsar SQL Getting Started

It is super easy to get started on querying data in Pulsar.

要求

  1. Pulsar distribution
    • 如果你没有安装Pulsar，请参考安装Pulsar
  2. Pulsar 内置连接器
    • 如果你没有安装内置连接器，请参考安装内置连接器

首先，启动Pulsar独立集群：

./bin/pulsar standalone

下一步，启动一个Pulsar SQL worker：

./bin/pulsar sql-worker run

在Pulsar独立集群和SQL worker完成初始化之后，运行SQL CLI：

./bin/pulsar sql

您现在可以开始输入一些SQL命令：

当前，Pulsar中没有我们可以查询的数据。让我们启动内置连接器* DataGeneratorSource *来导入一些模拟数据供我们查询：

./bin/pulsar-admin source create --tenant public --namespace default --name generator --destinationTopicName generator_test --source-type data-generator

然后，将有一个可以在"public/default"命名空间查询的主题：

现在，您有一些模拟数据可供查询和使用！

如果您想尝试获取一些自己的数据，可以编写一个简单的生产者将自定义数据写入Pulsar。

例如:

public class TestProducer {

    public static class Foo {
        private int field1 = 1;
        private String field2;
        private long field3;

        public Foo() {
        }

        public int getField1() {
            return field1;
        }

        public void setField1(int field1) {
            this.field1 = field1;
        }

        public String getField2() {
            return field2;
        }

        public void setField2(String field2) {
            this.field2 = field2;
        }

        public long getField3() {
            return field3;
        }

        public void setField3(long field3) {
            this.field3 = field3;
        }
    }

    public static void main(String[] args) throws Exception {
        PulsarClient pulsarClient = PulsarClient.builder().serviceUrl("pulsar://localhost:6650").build();
        Producer<Foo> producer = pulsarClient.newProducer(AvroSchema.of(Foo.class)).topic("test_topic").create();

        for (int i = 0; i < 1000; i++) {
            Foo foo = new Foo();
            foo.setField1(i);
            foo.setField2("foo" + i);
            foo.setField3(System.currentTimeMillis());
            producer.newMessage().value(foo).send();
        }
        producer.close();
        pulsarClient.close();
    }
}

接下来，你应该能够查询到刚写入的数据。
